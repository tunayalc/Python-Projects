# Python-Projects

Python ile farklı problem alanlarında geliştirdiğim bağımsız script ve mini araçlar

## Genel Bakış

`Python-Projects`, tek bir ürün fikrine bağlı olmayan; veri işleme, metin işleme, küçük otomasyonlar, istatistik, GUI ve API odaklı script'lerden oluşan bir çalışma alanı. Repo, Python'u farklı yönleriyle kullandığım dönemin geniş bir kesitini temsil ediyor.

## Kapsanan Alanlar

- kaynakça ve metin dönüştürme
- temel araç geliştirme
- istatistiksel değerlendirme
- finite state machine analizi
- doğal dil işleme denemeleri
- GUI tabanlı algoritma çalışmaları
- dış kaynaktan veri toplama

## Dosya Bazlı İçerik

| Dosya | Alan | Ne Yapıyor |
| --- | --- | --- |
| `bibtex_converter.py` | metin dönüştürme | farklı referans formatlarını BibTeX mantığına dönüştürmeye çalışıyor |
| `calculator.py` | temel araç | Python ile hesap makinesi akışı kuruyor |
| `data_science.py` | veri analizi | veri odaklı işlem mantığını deniyor |
| `distribution_evaluator.py` | istatistik | dağılım analizi ve kıyaslama mantığı |
| `fsm_analyzer.py` | durum makineleri | state transition mantığını inceleyen çalışma |
| `istiklal_marsi.py` | metin işleme | içerik ve string bazlı deneysel çalışma |
| `knn.py` | GUI + algoritma | KNN mantığını arayüz üzerinden işliyor |
| `lemma.py` | NLP | kelime normalizasyonu yaklaşımı |
| `pulling_data_from_reddit.py` | API / veri toplama | Reddit üzerinden veri çekme mantığı |
| `txt.py` | yardımcı araç | metin dosyaları üzerinde küçük işlem akışları |
